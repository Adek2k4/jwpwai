# Laboratoria JWPWAI
To repozytorium do zajęć Języki wysokiego poziomu w aplikacjach internetowych w ANS Elbląg. Główny projekt jest w folderze AdrianLewandowski21257 i zawiera laboratoria od prostego serwera Node.js oraz zadań z worker threads i obserwacji plików, przez aplikację Express z Pug, po projekty React i Next.js z widokami społeczności i posiłków. Strona startowa w katalogu głównym prowadzi do wdrożeń wybranych aplikacji.
Link do wersji GitHub Pages: https://adek2k4.github.io/jwpwai/
